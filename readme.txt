Standout Content adds an option to mark a post as standout for Google News.

== Description ==

Google announced the new standout tag for Google News publishers. If you put the tag in the HTML header of one of your articles, Google News may show the article with a ‘Featured’ label on the Google News homepage and News Search results. Please see the official post:
<a href="http://googlenewsblog.blogspot.com/2011/09/recognizing-publishers-standout-content.html">http://googlenewsblog.blogspot.com/</a>

This plugin adds a simple checkbox to each post where you can define if it stands out. 

== Screenshots ==

1. Just check or uncheck the checkbox below your post editor.

== Installation ==

1. Upload the whole folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. You're done. Edit your posts and check the standout-checkbox.
